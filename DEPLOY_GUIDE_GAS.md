# Google Apps Script 배포 가이드 🚀

이 프로젝트는 **Google Apps Script(GAS)**를 백엔드 서버로 사용합니다. 아래 순서대로 스크립트를 배포하고 URL을 발급받아야 합니다.

## 1단계: 구글 스프레드시트 생성
1.  브라우저에서 [sheet.new](https://sheet.new)에 접속하여 새 스프레드시트를 만듭니다.
2.  제목을 `Stock Data Backend` 등으로 변경합니다. (이 시트는 데이터 계산용으로 사용됩니다)

## 2단계: 스크립트 작성
1.  시트 메뉴에서 **확장 프로그램 (Extensions) > Apps Script**를 클릭합니다.
2.  `Code.gs` 파일의 내용을 모두 지우고, 프로젝트 폴더의 `gas_backend.js` 내용을 복사해서 붙여넣습니다.
3.  `Ctrl + S`를 눌러 저장합니다. (프로젝트 이름은 마음대로 설정)

## 3단계: 웹 앱으로 배포 (중요!)
1.  우측 상단의 **배포 (Deploy) > 새 배포 (New deployment)**를 클릭합니다.
2.  **유형 선택**: 톱니바퀴 아이콘 -> **웹 앱 (Web App)** 선택.
3.  **설정**:
    *   **설명**: `v1` 등 입력.
    *   **다음 사용자 권한으로 실행 (Execute as)**: **나 (Me)** (기본값 유지).
    *   **액세스 권한 (Who has access)**: **모든 사용자 (Anyone)** 👈 **가장 중요합니다! 이것을 선택해야 웹사이트에서 접근 가능합니다.**
4.  **배포 (Deploy)** 버튼 클릭.
5.  (권한 승인 팝업이 뜨면 '권한 검토' -> 계정 선택 -> '고급' -> '이동(안전하지 않음)' -> '허용' 순으로 진행합니다.)

## 4단계: URL 연결
1.  배포가 완료되면 **웹 앱 URL (Web App URL)**이 나옵니다. (`https://script.google.com/...` 형식)
2.  이 URL을 복사합니다.
3.  `Stock Data Collector` 웹사이트를 열고, 설정 패널에 이 URL을 붙여넣고 저장합니다.

이제 설정이 끝났습니다! 정적 웹사이트에서 데이터를 불러올 수 있습니다. 🎉
